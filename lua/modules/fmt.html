Module fmt

Functions for formatting strings.

Usage:

    local fmt = require "fmt"

Functions

number (number)
Converts an integer into a human readable string.

The conversion rounds to the nearest integer and inserts commas every three places.

Parameters:

  • number number Number to convert.

Returns:

    string The converted string.

Usage:

    s = fmt.number(60000000) -- Converts to "60,000,000"
credits (credits)
Converts a number of credits to a string (e.g. "10.00 k¢").

The conversion method is the same as in C credits2str function, but locked to using 2 decimal places.

Parameters:

  • credits number Number of credits.

Returns:

    string The converted string.

Usage:

    misn.setReward(fmt.credits(100000))
tonnes (tonnes)
Converts a number of tonnes to a string.

DO NOT USE THIS WITHIN SENTENCES. Relying on it for that use-case can lead to sentences which cannot be properly translated. Use this function ONLY for cases where a number of tonnes is shown alone.

Parameters:

  • tonnes number Number of tonnes.

Returns:

    string The converted string.
f (str, tab)
String interpolation function.

Inspired by f-strings, but more similar to Python's str.format() and simplified. Prefer this over string.format because it allows translations to change the word order.

Parameters:

  • str string Format string which may include placeholders of the form "{var}" or "{var:6.3f}" (where the expression after the colon is any directive string.format understands). A var which is a number is treated as a number; all other var definition is treated as a string. This value is used to index the argument table.
  • tab table Argument table.

Usage:

  • fmt.f(_("Land on {pntname} ({sysname} system)"), {pntname=returnpnt:name(), sysname=returnsys:name()})
  • fmt.f(_("As easy as {1}, {2}, {3}"), {"one", "two", "three"})
  • fmt.f(_("A few digits of pi: {1:.2f}"), {math.pi})

Module system

Lua system module.

This module allows you to use the Star Systems from Lua.

Typical example would be something like: 

 cur = system.get() -- Gets current system
 sys = system.get( "Gamma Polaris" )

get (param)
Gets a system.

Behaves differently depending on what you pass as param:
- string : Gets the system by raw (untranslated) name.
- planet : Gets the system by planet.

Parameters:

  • param string or Planet Read description for details.

Returns:

    System System matching param.

Usage:

  • sys = system.get( p ) -- Gets system where planet 'p' is located.
  • sys = system.get( "Gamma Polaris" ) -- Gets the system by name.

name (s)
Returns the system's translated name.

This translated name should be used for display purposes (e.g. messages). It cannot be used as an identifier for the system; for that, use system.nameRaw() instead.

Parameters:

  • s System System to get the translated name of.

Returns:

    string The translated name of the system.

Usage:

    name = sys:name() -- Equivalent to `_(sys:nameRaw())`
nameRaw (s)
Returns the system's raw (untranslated) name.

This untranslated name should be used for identification purposes (e.g. can be passed to system.get()). It should not be used directly for display purposes without manually translating it with _().

jumpDist (s[, dest[, hidden=false[, known=false]]])
Gets jump distance from current system, or to another.

Parameters:

  • s System Starting system.
  • dest string or System System to get distance to. If nil or unspecified, gets distance to the current system. (optional)
  • hidden boolean Whether or not to consider hidden jumps. (default false)
  • known boolean Whether or not to consider only jumps known by the player. (default false)

Returns:

    number or nil Number of jumps to system, or nil if there is no route to the system with the given parameters.

Usage:

  • d = sys:jumpDist() -- Distance from sys to current system.
  • d = sys:jumpDist("Draygar") -- Distance from sys to system Draygar.
  • d = sys:jumpDist(another_sys) -- Distance from sys to another_sys.
jumpPath (s[, dest[, hidden=false[, known=false]]])
Gets jump path from current system, or to another.

Parameters:

  • s System Starting system.
  • dest string or System System to get path to. If nil or unspecified, gets path to the current system. (optional)
  • hidden boolean Whether or not to consider hidden jumps. (default false)
  • known boolean Whether or not to consider only jumps known by the player. (default false)

Returns:

    {Jump,...} or nil Table of jumps, or nil if there is no route to the system with the given parameters.

Usage:

  • jumps = sys:jumpPath() -- Path from sys to current system.
  • jumps = sys:jumpPath("Draygar") -- Path from sys to Draygar.

presences (s)
Returns a table indicating faction presence levels.

The table returned is indexed by raw (untranslated) faction name and indicates the corresponding presence number. Factions with no presence in the system are not included in the table (meaning that attempting to index the table with factions that are not in the system will give you nil instead of 0, and looping through the table will yield only factions that have some amount of presence in the system).

Parameters:

  • s System System to get the faction presences of.

Returns:

    table A table indicating presences, indexed by raw (untranslated) faction name. See above for a complete explanation.

See also:

    presence

Usage:

  • if sys:presences()["Empire"] then
   -- The system has Empire ships.
end
  • for k, v in pairs(sys:presences()) do
   print(string.format(_("%s presence is %d."), _(k), v))
end

presence (sys, id)
Gets a presence amount in the system.

Parameters:

  • sys System System to get presence level in.
  • id Faction or string What to get the presence level of. Can be a faction to get the presence of one faction; "all" to get the total presence of all factions; "friendly" to get the total presence of all friendly factions; "hostile" to get the total presence of all hostile factions; or "netural" to get the total presence of all neutral factions.

Returns:

    number The presence level in sys (absolute value).

See also:

    presences

Usage:

  • p = sys:presence(f) -- Gets the presence of a faction f
  • p = sys:presence("all") -- Gets the sum of all the presences
  • if sys:presence("friendly") > sys:presence("hostile") then
   -- System has more friendly presences than hostile presences.
end

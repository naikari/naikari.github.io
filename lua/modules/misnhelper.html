Module misnhelper

Generic mission-helping functions.

Usage:

    local mh = require "misnhelper"

Functions

anyMissionActive (names)
Wrapper for player.misnActive that works on a table of missions.

Parameters:

  • names table Table of names of missions to check

Returns:

    boolean true if any of the listed missions are active, or false otherwise.

Usage:

    if mh.anyMissionActive({"Cargo", "Cargo Rush"}) then
showWinMsg ([congrats])
Shows a player message informing of a mission success.

Note: this does not actually end the mission. It should be followed up with a misn.finish(true) call.

Parameters:

  • congrats string Congratulation message for winning. (optional)

Usage:

    mh.showWinMsg(_("You delivered the cake!"))
showFailMsg ([reason])
Shows a player message informing of a mission failure.

Note: this does not actually end the mission. It should be followed up with a misn.finish(false) call.

Parameters:

  • reason string The reason the player failed the mission. (optional)

Usage:

    showFailMsg(_("You failed to deliver the cake!"))
